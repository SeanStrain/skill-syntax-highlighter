# Skill README

Syntax highlighter and snippets to skill files in visual studio code.

## Features

![](https://github.com/herbertagosto/code-skill/blob/main/images/skill.gif)

### Folding

```skill
;region REGION NAME
your blocks of code...

;endregion
```

### Snippets

```skill
region
defun
procedure
let
when
unless
case
if
if else
foreach
for
setof
mapcar lambda
```
